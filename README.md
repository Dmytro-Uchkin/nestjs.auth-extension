## Description

Official NestJS on Authentication and Authorization

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Running UI App

[UI application](https://github.com/Dmytro-Uchkin/nestjs.auth-extension-ui)

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
